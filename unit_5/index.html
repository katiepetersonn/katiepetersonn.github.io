What is Coding?

Coding provides a means of communicating with your computer. When you write code, it tells the computer what to do.

A computer is essential a collection of on/off switches (transistors). Modern computers contain millions or even billions of transistors, which means an unimaginably large number of combinations.

Binary code is the representation of transistor combinations as ones and zeros, where each digit represents one transistor. Binary code is clustered together in groups of eight to form bytes, e.g 11011010.

Writing a computer program by typing out billions of ones and zeros would be tediously time-consuming, which is where various programming languages enter the picture. A program is simply a text file, written in a certain programming language, and the code inside a program file is called source code.

Some of the most popular languages include JavaScript, Python, Ruby, SQL, PHP, C, C++, Java, Object-C and Perl.

Girl Crush #1: Margaret Hamilton

Margaret Hamilton begins our Girl Crush feature, where we share with you who we are crushing on within the technology insustry.

There are numerous names that most people associate with humans landing on the mooon, Buzz Aldrin, Neil Armstrong, but Margaret Hamilton is not usually among them. Yet this pioneering women was the lead software developer for Apollo's flight.

For Hamilton and her team, programming consisted of punching holes in cards to execute certain processes. The cards would be run through a giant Honeywell mainframe computer that would stimulate the Apollo lander, making sure everything functioned as planned.

Many of the earliest and pioneering programmers were women, learning hands-on to do things that had never been done before.

Like female coders in today's diversity-challenged tech industry, Hamilton was an outlier. It might be a surprise to some that one of the founding fathers of software development was, in fact, a women.

Thanks to Hamilton and the work she led, notions of what humanity could do, and be, changed not only the stratosphere but computer science. Software engineering, a concept she pioneered, has found its way from the moon to nearly every human endevour.
